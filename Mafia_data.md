## 마피아 게임 정리
__1. 클래스__
- ***User***: __영구적으로__ 유지되는 사용자의 데이터
  - *Id*: 사용자의 출석번호
  - *PW*: 사용자의 로그인 코드
  - *Name*: 사용자의 이름
  - *Level*: 사용자의 레벨
  - *Exp*: 사용자의 경험치
  - *Team*: 이벤트에서의 팀
- ***Game***: __일시적으로__ 만들어지는 한 게임의 데이터
- ***Mission***: __영구적으로__ 유지되는 미션(임무)의 데이터
- ***Role***: __영구적으로__ 유지되는 능력의 데이터(플레이어에게 랜덤 배정)
- ***Player***: 게임 시작시 __일시적으로__ 만들어지는 플레이어의 데이터
